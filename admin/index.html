GitHub Pages Deployment Guide

Follow these steps to publish or update your Neilz Network site.

How updates work: Edit content in the admin panel → Download services.json → Replace the file in your repository → Push to GitHub. The site updates automatically within ~60 seconds.
1

Create a GitHub Account & Repository

Go to github.com, create a free account, then click New Repository. Name it something like neilznetwork. Set it to Public and leave everything else default.

2

Install Git & Upload Your Files

Download Git from git-scm.com. Then open a terminal in your site folder and run:

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main

Replace YOUR_USERNAME and YOUR_REPO with your real values.

3

Enable GitHub Pages

In your repository, go to Settings → Pages. Under Source, select Deploy from a branch, choose main branch and / (root) folder. Click Save.

Your site will be live at: https://YOUR_USERNAME.github.io/YOUR_REPO/

4

Connect a Custom Domain (Optional)

In Settings → Pages → Custom domain, enter your domain (e.g. neilznetwork.com). Then update your domain's DNS:

A     @    185.199.108.153
A     @    185.199.109.153
A     @    185.199.110.153
A     @    185.199.111.153
CNAME www  YOUR_USERNAME.github.io

DNS changes take up to 24 hours. Enable Enforce HTTPS once the domain is verified.

5

Updating Your Content

Whenever you make changes in this admin panel:

1. Click "Download services.json" above
2. Replace /data/services.json in your project folder
3. Run these commands:

git add data/services.json
git commit -m "Update services content"
git push

GitHub Pages will rebuild and your changes will be live within about a minute.

6

Set Up the Contact Form (Optional but Recommended)

The contact form uses mailto: by default. For a real form submission, sign up free at formspree.io, create a form, and replace the form's action attribute in index.html:

<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
