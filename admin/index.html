Steps to Go Live
1

Download the JSON

Click the button above.

2

Replace the file

Move the downloaded file into your project's data/ folder, replacing the existing one.

3

Push to GitHub

git add data/services.json
git commit -m "Update site content"
git push
4

Hard refresh

Wait ~60 seconds, then press Ctrl+Shift+R (or Cmd+Shift+R) to see changes.
